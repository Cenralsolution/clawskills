# OpenClaw Skills Workspace Rules

These rules apply to any prompt in this workspace.

## 1. Scope
Any prompt in this workspace is about **OpenClaw**.

## 2. Skills Definition
When talking about skills, skills for OpenClaw are meant. Skills are deployed and to be deployed to https://clawhub.ai/

## 3. New Skill Development Process
If a new skill should be created, follow this process:
- Check requirements with a requirements engineer
- Design a solution with a solution architect (zero trust and security-first approach)
- Develop it with a developer
- Conduct testing with a tester
